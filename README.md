### 표기
```
$ : command 혹은 terminal에서 실행($ = shell prompt)
>>> : python에서 실행
🐸 : 순한맛, python
😲 : 중간맛, python + 컴퓨터 배경 지식
🥵 : 매운맛, python + 컴퓨터 배경 지식 + 실제 코드

중간맛과 매운맛은 저도 잘 몰라서 공부하려고 적어둡니다
```
